# "noir"

#### Description
Side-View John Wick-Inspired PICO-8 prototype. 
The player will press a single button to do gun-fu counters on attacking enemies.
It will be a power-trip in which the player character cannot be hurt, meaning there is no fail-state.

#### Character
The player character will wear a dark trenchcoat to distinguish him from the enemies.
He will use a semi-auto sidearm.

#### Style
Visual style is inspired by the 1960s.
Muted palette consisting of shades and low saturation colors.
Blood will be colored with high saturation for contrast.

16x24px
